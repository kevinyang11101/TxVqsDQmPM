# 物业管理系统 java132

## 项目概述

物业管理系统是一款基于前后端分离架构的信息管理系统，旨在为物业公司提供全面的信息化解决方案。系统采用 Spring Boot 和 Vue.js 技术栈开发，实现了物业管理的高效与便捷。

## 技术栈

- 前端：Vue.js
- 后端：Spring Boot

## 功能模块

### 物业管理端

- **首页**
- **员工管理**
- **角色管理**
- **权限管理**
- **栋数管理**
- **单元管理**
- **房屋列表**
- **车位管理**
- **业主管理**
- **电费管理**
- **水费管理**
- **停车管理**
- **投诉管理**
- **维修管理**
- **公告管理**

### 业主端

- **首页**
- **投诉管理**
- **缴费管理**
  - 我的电费
  - 我的水费
  - 我的停车费
- **维修管理**
- **公告管理**

## 系统角色

- 物业管理员
- 业主

## 运行环境

- 前端：现代浏览器（推荐使用 Chrome）
- 后端：Java 1.8 或更高版本

## 部署步骤

1. 准备相应的服务器环境（Java、数据库等）
2. 部署后端服务（Spring Boot 应用）
3. 部署前端应用（Vue.js 应用）
